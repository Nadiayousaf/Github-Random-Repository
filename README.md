# 🌐 GitHub Random Repository Finder

A sleek, responsive web application that leverages the GitHub API to help developers and tech enthusiasts discover random public repositories. Built using modern web technologies and styled with Bootstrap for a seamless user experience across all devices.

## 🚀 Features

* **Dynamic Fetching:** Instantly fetches random public repositories using the official GitHub API.
* **Detailed Stats:** Displays repository names, descriptions, primary languages, star counts, and fork metrics.
* **Responsive Design:** Utilizes Bootstrap 5 for a mobile-first, clean, and modern user interface.
* **Error Handling:** Safe handling for API rate limits and network connection issues.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+, Fetch API)
* **Styling Framework:** Bootstrap 5
* **API Integration:** GitHub REST API

## 📦 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   ```

2. **Navigate to the project directory:**
   ```bash
   cd github-random-repo-finder
   ```

3. **Run the application:**
   Simply open `index.html` in your favorite web browser.
